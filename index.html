NumPy: cálculo numérico con arreglos

NumPy es la base del ecosistema científico de Python: proporciona arreglos multidimensionales y operaciones vectorizadas eficientes para cálculo numérico [web:17].

Conceptos clave

  • ndarray Estructura central: un arreglo n-dimensional homogéneo.
  • Vectorización Operar sobre arreglos completos sin bucles explícitos de Python.
  • Broadcasting Reglas para operar arreglos de formas diferentes de manera automática.

Matplotlib: gráficos estáticos en 2D

Matplotlib es la librería estándar de gráficos en Python para crear figuras estáticas y altamente personalizables [web:11][web:15].

Configuración básica y tipos de gráficos

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

plt.figure(figsize=(6, 4))
plt.plot(x, y, label="sin(x)")
plt.xlabel("x")
plt.ylabel("y")
plt.title("Función seno")
plt.legend()
plt.grid(True)
plt.show()

Plotly Graph Objects: mayor control

import plotly.graph_objects as go
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

fig = go.Figure()
fig.add_trace(go.Scatter(x=x, y=y, mode="lines", name="sin(x)"))
fig.update_layout(
    title="Función seno interactiva",
    xaxis_title="x",
    yaxis_title="sin(x)"
)
fig.show()
